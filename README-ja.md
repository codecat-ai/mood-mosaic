# Mood Mosaic

[English](README.md) | [简体中文](README-zh.md) | [日本語](README-ja.md)

Mood Mosaic は、学生、作り手、リモートワーカー、ファシリテーター向けのローカルファーストな気分、エネルギー、集中度のジャーナルです。アカウントやクラウド同期なしで個人的な振り返りを残せます。

## 課題

多くの気分記録アプリはアカウントやクラウド保存に依存していたり、診断を連想させる表現を含んでいたりします。習慣トラッカーは数値中心になりがちで、スプレッドシートは短い日々の文脈を書くには扱いにくいです。Mood Mosaic はデータをブラウザーに保存し、小さな日次記録を読みやすいカレンダーモザイクと短い要約に変えます。

## 機能

- 選択した日付に気分、エネルギー、集中度、短いメモを追加または更新。
- 日付順のカラーモザイク、週次レビューのような平均、気分の件数、連続記録日数を表示。
- ジャーナル、コーチ、チェックイン用に簡潔な振り返り要約をコピー。
- ブラウザーから JSON バックアップをエクスポート、インポート。
- 無効なデータや旧 schema のデータを扱ってもクラッシュしない検証。
- 本番ビルド後は完全にクライアント側で動作。

## ソースからのインストール

Mood Mosaic はパッケージレジストリでは公開していません。ソースチェックアウトを使ってください。

```bash
git clone <repository-url>
cd mood-mosaic
npm ci
```

## クイックスタート

```bash
npm run dev
```

Vite が表示するローカル URL を開きます。今日の日付または過去の記録日を選び、気分、エネルギー、集中度、メモを入力し、モザイクと要約パネルで最近の傾向を確認します。

## 例

記録例:

```json
{
  "date": "2026-05-21",
  "mood": "focused",
  "energy": 4,
  "focus": 5,
  "note": "Finished the project draft",
  "schemaVersion": 1
}
```

バックアップ例:

```json
{
  "schemaVersion": 1,
  "entries": []
}
```

## 設定

アカウント、サーバー、クラウド設定はありません。データはブラウザー LocalStorage の `mood-mosaic:journal` に保存されます。持ち運べるバックアップやブラウザー移行が必要な場合は、JSON を定期的にエクスポートしてください。

## 開発

```bash
npm run lint
npm run typecheck
npm test -- --run
npm run build
```

コアモデルは `src/journal.ts`、分析は `src/summary.ts`、ストレージアダプターは `src/storage.ts`、React UI は `src/App.tsx` にあります。

## テスト

テストは検証と upsert、分析とモザイクの並び順、ストレージのインポートとエクスポート、旧 schema の正規化、日付ごとの編集、UI のスモークフローを対象にしています。振る舞いを変える前にテストを書いてください。
